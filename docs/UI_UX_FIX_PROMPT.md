# 오피스 UI 레이아웃 버그 리포트 및 전면 수정 지시서 (Prompt for Claude)

## 📌 현재 심각한 레이아웃 문제 진단 (Report)

클로드가 1차적으로 디자인을 수정했으나, 레이아웃이 완전히 붕괴된 상태입니다. 스크린샷 렌더링 결과 다음과 같은 3가지 치명적인 문제가 발견되었습니다.

1. **하단 모눈종이(그리드) 배경이 남아있는 원인:**
   - **분석:** 전체 `<body>` 또는 최상위 래퍼(Wrapper) 컨테이너의 배경을 수정하지 않고, 상단 작업 구역의 `div`에만 베이지색 배경을 적용했기 때문입니다. 또는 컨테이너의 높이(`height: 100vh` 등) 설정이 어긋나서, 콘텐츠가 없는 하단 빈 공간에 기존 사이버펑크 스타일의 어두운 그리드 배경이 그대로 노출되고 있습니다.
2. **화면 우측 잘림 및 사이드바 증발:**
   - **분석:** 메인 오피스 영역의 너비가 제어되지 않고 팽창하여 화면 우측이 잘리며, 이전 스크린샷에 있던 우측 정보 패널(Agent Status 사이드바)이 화면 밖으로 밀려나거나 삭제되어 렌더링되지 않고 있습니다.
3. **UI 요소들의 심각한 오버랩(겹침):**
   - **분석:** 
     - 상단 'Planner', 'Architect' 등 역할명 박스(반투명)와 새로 만든 갈색 책상 박스들이 겹쳐서 출력됩니다.
     - 중간에 위치한 'Desk 1' 아래에 정체불명의 가구가 겹쳐 있습니다.
     - 우측 하단 아바타(이모지) 카드들이 여전히 엉뚱한 위치(소파와 겹침)에 공중에 떠 있습니다. Flexbox나 Grid가 올바르게 작동하지 않고, 무분별한 `position: absolute`가 얽혀있는 것으로 추정됩니다.

## 🛠 클로드를 위한 긴급 수정 및 개편 지시사항 (Action Items)

현재 코드를 완전히 뜯어고쳐서, 브라우저 크기에 맞게 딱 떨어지는 **'깔끔한 2D 탑다운 오피스 뷰'**를 완성해 주세요.

### 1. 전역 배경색 통일 및 그리드 완전 삭제
- `/styles/main.css`에서 기존 다크 모드의 모눈종이 배경(어두운 파란색 + 격자무늬)을 **완전히 삭제**하세요.
- `body`, `#app`, `main` 등 최상위 계층의 배경색을 레퍼런스와 같은 따뜻한 톤(예: `#F4EFE6` 같은 베이지/아이보리 우드 톤)으로 **화면 100% 전체에** 덮어씌워 일관성을 만드세요.

### 2. 레이아웃 리부트 (오피스와 사이드바 영역의 명확한 분배)
- 화면 전체를 Flexbox 요소로 설정하여 투-컬럼(Two-column) 레이아웃을 확립하세요.
- **Left/Center (오피스 뷰 - 약 75% 너비):** 데스크, 라운지, 소파 등이 배치되는 메인 사무 공간.
- **Right (상태 대시보드 - 약 25% 너비):** 잘려나간 우측 사이드바를 다시 복구하고, 에이전트 상태를 보여주세요. (오피스 뷰와 절대 겹치지 안도록 스크롤과 영역을 분리!)

### 3. 무질서한 겹침(Overlap) 전면 제거 및 데스크 정렬
- **Work Zone (상단 파티션):** 갈색 책상 위로 반투명한 박스가 겹치지 않도록 구조를 합치세요. 그냥 깔끔하게 갈색 책상을 하나 만들고, 그 중앙에 'Planner' 등의 텍스트와 컴퓨터 모니터 이모지(`💻`)를 배치하는 식으로 직관적으로 바꾸세요. 
- **Middle Zone (중앙 데스크):** 'Desk 1, 2, 3'이 서로 겹치지 않게 Flex 줄바꿈(`flex-wrap`)이나 마진을 올바르게 주세요.
- **Lounge Zone (우측 하단):** 에이전트 아바타들이 소파나 책상 위치에 자연스럽게 소속되도록 컨테이너 안에 넣으세요. 허공에 겹쳐 뜨는 `position: absolute` 설정은 모두 없애거나 상위 컨테이너 기준으로 제한하세요.

### 4. 시각적 완성도 디테일 업
- 단순히 갈색 박스 4개를 이어 붙인 형태가 아니라, 약간의 보더 반경(`border-radius`), 부드러운 박스 그림자(`box-shadow`) 등을 추가하여 '위에서 내려다본 책상이나 소파'라는 느낌을 퀄리티 있게 살려주세요.

**모든 수정을 마친 후, 왜 화면이 짤리고 모눈종이가 떴었는지 그 원인을 고쳤다고 설명하고, 수정된 CSS/HTML 구조를 반영해 주세요.**
